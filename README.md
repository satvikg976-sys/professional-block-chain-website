# 🚀 Satvik Solutions – Blockchain & Web3 Business Website

## 📌 Project Overview

Satvik Solutions is a modern and responsive business website developed using HTML, CSS, and JavaScript. The website represents a Blockchain and Web3 technology company that provides blockchain development services, smart contract solutions, Web3 applications, decentralized application (DApp) development, enterprise blockchain solutions, and professional blockchain training.

The website is designed with a professional dark theme, smooth animations, responsive layouts, interactive UI elements, and modern design principles to provide an excellent user experience.

# 📁 Installation

1. Download or Clone the repository.

2. Open the project folder.

3. Open the project using Visual Studio Code.

4. Install the Live Server Extension (optional).

5. Right-click on **index.html**.

6. Click **Open with Live Server**.
